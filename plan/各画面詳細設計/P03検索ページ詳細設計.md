# P03 検索ページ詳細設計

## 1. 画面概要

記事のキーワード検索、および検索結果一覧を表示する。検索クエリがない場合は最新記事等を表示する。

## 2. 必要な表示データ

- **検索結果記事**: クエリにマッチした記事リスト。
- **最新記事（フォールバック）**: 検索結果が0件または検索前の場合に表示。

## 3. I/O (API フォーマット)

### 3.1. 記事検索

- **メソッド**: `GET`
- **エンドポイント**: `/api/articles`
- **Query Parameters**:
  - `q`: 検索キーワード（タイトル、要約、本文等を対象）
  - `page`: ページ番号 (Default: 1)
  - `limit`: 1ページあたりの件数 (Default: 12)
- **Response Example**:

```
{
"articles": [
  {
    "article_id": 10,
    "title": "TypeScript入門",
    // ...
  }
],
"total": 100,
"page": 1,
"limit": 12
}
```

### 3.2. 最新記事一覧取得（フォールバック用）

- **メソッド**: `GET`
- **エンドポイント**: `/api/articles`
- **Request**: Query `?type=latest` or Body `{"type": "latest"}`
- **Response**: P01と同様。
