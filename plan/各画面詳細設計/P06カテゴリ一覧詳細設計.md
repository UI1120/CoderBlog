# P06 カテゴリ一覧詳細設計

## 1. 画面概要

全カテゴリの一覧を表示するタイル状のグリッド画面。各カテゴリをクリックすると、そのカテゴリで検索（`/search?q={CategoryName}`）へ遷移する。

## 2. 必要な表示データ

- **カテゴリ一覧**: ラベル、ID、紐づく記事数（任意）。

## 3. I/O (API フォーマット)

### 3.1. カテゴリ一覧取得

- **メソッド**: `GET`
- **エンドポイント**: `/api/categories`
- **Response Example**:

```json
[
  {
    "id": 1,
    "label": "Web Development",
    "articleCount": 15
  },
  {
    "id": 2,
    "label": "Robotics",
    "articleCount": 8
  }
]
```
