# P01 トップページ詳細設計

## 1. 画面概要

サイトのランディングページ。ヒーローセクション、活動紹介（About）、プロジェクトハイライト、最新記事、お知らせを表示する。
ユーザーへの第一印象を決定づけるため、動的かつ視覚的にリッチなデザインを提供する。

## 2. 必要な表示データ

- **Latest Articles**: 最新の記事一覧（カルーセル表示用）。
- **Project Highlights**: 注目の活動プロジェクト等（カルーセル表示用）。
- **Hero/About**: 静的コンテンツ（`siteConfig.ts` 等から読み込み）。

## 3. I/O (API フォーマット)

### 3.1. 最新記事一覧取得

- **メソッド**: `GET`
- **エンドポイント**: `/api/articles`
- **Request**: Query `?type=latest` or Body `{"type": "latest"}`
- **Response Example**:

```json
[
  {
    "id": 1,
    "title": "ロボット制御の基礎",
    "summary": "ROS2を用いた制御...",
    "thumbnail": "https://...",
    "published_date": "2026-01-20",
    "category": "Robotics",
    "writer": "Yamada"
  }
]
```

### 3.2. プロジェクトハイライト取得

- **メソッド**: `GET`
- **エンドポイント**: `/api/articles`
- **Request**: Query `?type=project_highlights` or Body `{"type": "project_highlights"}`
- **Response Example**:

```json
[
  {
    "id": 5,
    "title": "Project Alpha",
    "summary": "Key project article...",
    "thumbnail": "..."
  }
]
```
