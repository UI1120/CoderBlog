# P00 共通詳細設計

## 1. 概要

Webサイト全体で共通して使用されるコンポーネント（ヘッダー、フッター等）および共通のAPI通信仕様について定義する。

## 2. 実装コンポーネント (src/P00_common)

- **Header**: サイト全体のナビゲーション、検索バー、ハンバーガーメニュー（SP）。
- **Footer**: サイトマップ、著作権表示、SNSリンク。
- **UI Components**: `Button`, `Input`, `Pagination`, `CardMatrix` 等の再利用可能なUIパーツ。
- **Hooks**: `useMobile` (SP判定) 等。

## 3. I/O (API フォーマット)

### 3.1. ヘッダー用メニュー項目取得

ヘッダーのドロップダウンメニューに表示する項目リストを取得する。

#### プロジェクト一覧

- **メソッド**: `GET`
- **エンドポイント**: `/api/header/projects`
- **Response Example**:

```json
[
  { "label": "Game Dev", "href": "/project?pid=1" },
  { "label": "Robot", "href": "/project?pid=2" }
]
```

#### カテゴリ一覧

- **メソッド**: `GET`
- **エンドポイント**: `/api/header/categories`
- **Response Example**:

```json
[
  { "label": "Frontend", "href": "/search?q=Frontend" },
  { "label": "Infrastructure", "href": "/search?q=Infrastructure" }
]
```

#### タグ一覧

- **メソッド**: `GET`
- **エンドポイント**: `/api/header/tags`
- **Response Example**:

```json
[
  { "label": "React", "href": "/search?q=React" },
  { "label": "Python", "href": "/search?q=Python" }
]
```

#### クリエイター一覧（ライター）

- **メソッド**: `GET`
- **エンドポイント**: `/api/header/writers`
- **Response Example**:

```json
[
  { "label": "Yamada", "href": "/creator?cid=1" },
  { "label": "Tanaka", "href": "/creator?cid=2" }
]
```

#### グループ一覧 (Option)

- **メソッド**: `GET`
- **エンドポイント**: `/api/header/groups`
