# P07 タグ一覧詳細設計

## 1. 画面概要

全タグの一覧を表示する画面。各タグをクリックすると検索ページへ遷移する。

## 2. 必要な表示データ

- **タグ一覧**: ラベル、ID、紐づく記事数（任意）。

## 3. I/O (API フォーマット)

### 3.1. タグ一覧取得

- **メソッド**: `GET`
- **エンドポイント**: `/api/tags`
- **Response Example**:

```json
[
  {
    "id": 1,
    "label": "React",
    "articleCount": 10
  },
  {
    "id": 2,
    "label": "Python",
    "articleCount": 5
  }
]
```
